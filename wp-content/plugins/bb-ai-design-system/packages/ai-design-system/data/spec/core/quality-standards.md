## Quality Standards

- **Images** — include `<img>` tags with descriptive alt text that captures the ideal image content and mood (e.g., `alt="Team of diverse professionals collaborating around a whiteboard in a modern office"`). The system sources appropriate images from alt text. Don't skip images because you lack a URL.
- **Icons and decorative elements** — use inline SVG, never emoji characters (unless the brief explicitly requests emoji). The `data-field-type="svg"` codec doesn't handle emoji.
